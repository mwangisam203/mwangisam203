## Featured Projects

### SafariDesk

A paid technical-content platform for African developers.

**Tech Stack:**  
FastAPI, PostgreSQL, Redis, Celery, React, Tailwind CSS, Docker, M-Pesa

**What it does:**

- Handles user authentication and protected routes
- Manages paid article access
- Supports M-Pesa subscription payments
- Uses background jobs for email and SMS tasks
- Includes reader and admin dashboard features
- Uses Docker for a cleaner development and deployment setup

**Core Concepts:**  
API design, authentication, subscriptions, payments, background jobs, database relationships, Docker, and deployment-ready architecture.

[View Repository](https://github.com/mwangisam203/safaridesk)

---

### FastAPI Blog Platform

A full-stack blog application built with FastAPI and PostgreSQL.

**Tech Stack:**  
FastAPI, PostgreSQL, SQLAlchemy, Alembic, JavaScript, Bootstrap, Jinja2

**What it does:**

- User registration and login
- Blog post creation, editing, and deletion
- Comments and likes
- User profiles
- Image uploads
- Password reset flow
- Pagination
- Health checks

**Core Concepts:**  
CRUD operations, authentication, relational databases, migrations, file uploads, templates, API structure, and backend testing.

[View Repository](https://github.com/mwangisam203/fastapi-blog-postgres)

### TranscriptsKE

A Kenyan academic transcript request and verification platform.

**Tech Stack:**  
FastAPI, PostgreSQL, SQLAlchemy, Alembic

**What it does:**

- Allows students to request academic transcripts
- Helps institutions manage transcript workflows
- Supports academic record verification
- Provides a foundation for student, institution, and employer-facing features

**Core Concepts:**  
Database design, request workflows, verification systems, backend architecture, and API development.

[View Repository](https://github.com/mwangisam203/TranscriptKe)
